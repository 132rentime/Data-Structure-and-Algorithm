## Trie（前缀树）

#### 什么是Trie字典树

- **多叉树，专门处理字符串**

- **时间复杂度和字典中条目的数量无关，若查询单词的长度为w，时间复杂度为O(w)**

  ![](Trie.png)

- 节点表示

  **当前结点是否代表一个单词的结尾**
  
  **每个结点有若干个指向下一个结点的指针**
  
  ```c++
  class Node {
      boolean isWord;
      Map<char, Node> next;
  };
  ```
